# Roblox Asset Browser — Privacy Policy

This repository hosts the privacy policy for the [Roblox Asset Browser](https://github.com) Chrome and Firefox extension.

## Live page

[Privacy Policy](https://rivrs-roblox.github.io/roblox-asset-browser-privacy/)

## About the extension

Roblox Asset Browser is a browser extension for Roblox developers. It lets you browse, search, upload and manage your assets (Decals, Images, Audio, Models, Animations, MeshParts) across personal and group accounts — directly from Chrome or Firefox, without the Creator Dashboard.

## Summary

The extension only accesses your Roblox session cookie (`.ROBLOSECURITY`) to authenticate API requests on your behalf. No data is collected, sold, or shared with any third party. All requests go directly to Roblox's official API endpoints.
